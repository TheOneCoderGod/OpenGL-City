Assignment 2 SkyPacks (from 2004)

The skyboxes below are from 2004. The 2005 versions (available here) were created using a different technique. The differences are:

  • These (2004) skybox images are 597x597 pixels. The 2005 skyboxes are 1024x1024 pixels. (Note that the texture loader currently scales all textures to a maximum size of 512x512 pixels. If you want to get the benefit of 1024x1024 pixel skybox faces, you'll need to modify the code in the texture loader in platform.cpp. Note that larger faces will be more taxing on the graphics hardware.)
  • The 2004 images were created by converting Bryce panoramas to the 6 cube faces. Bryce doesn't render a full 180 degree vertical sweep when creating a panorama, and as a result the top face has distortions at the North pole. This won't be apparent if you don't point the camera up too far.
  • Some of the 2004 skypacks use the same filenames as the 2005 sets. If you download and use any of these, be careful that you don't overwrite files from the 2005 set that you might be using.

Sky 21 is a simple grid that is useful to show how an undistorted image appears when used for a skybox (it just look like you're in a box).

Sky 22 shows that you can create simple gradients that suit as very basic skies (this was created with Photoshop), although there is still some additional darking at the corners in the projected skybox that give away the illusion.

Sky 42 is similar to sky 21, in that it was created using a simple grid, but the difference is that the grid in 42 was created as a panoramic image before it was converted into the 6 cube face images.
